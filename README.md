# Pygame Boilerplate

## Description

This is a [Pygame](https://www.pygame.org/docs/) game boilerplate.<br />

## Background

Run main.py in the root folder to start the program.<br />

## Testing

Install pytest>=7.0.0<br />
Run all tests with verbose output: pytest -v<br />
